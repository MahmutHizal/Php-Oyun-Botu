Php-Oyun-Botu
=========

Basit bir oyun botu. [Bilal İşler](http://bilalisler.net/)'in [PHP Oyun Botu Nasıl yapılır? - PHP BOT](http://bilalisler.net/php-oyun-botu-nasil-yapilir/) adlı yazısından esinlenerek yazdığım basit bir oyun botu.

####Gerekli paketler:
* [sunra/php-simple-html-dom-parser](https://github.com/sunra/php-simple-html-dom-parser)
* [lsolesen/php-template](https://github.com/lsolesen/php-template)


##Kurulum
---
* composer.json dosyasını composer aracılığıyla çalıştırıp gerekli paketleri indiriyoruz.
